# Event Registration Data Mining Project

## Problem Statement
The project aims to analyze event registration data to discover participant behavior patterns and improve event planning.

## Dataset
The dataset contains details of events, participants, and registrations.

## Objectives
- Analyze registration trends
- Cluster participants based on behavior
- Predict event attendance

## Tools & Technologies
- Python
- Pandas, NumPy, Scikit-learn
- MySQL / SQLite
- VS Code / Jupyter Notebook
